# huddle-ai
Huddle AI es un chatbot diseñado para responder preguntas relacionadas con la NFL de forma rápida, clara y precisa. Su objetivo es brindar información confiable sobre equipos, jugadores, estadísticas, reglas y eventos de la liga, funcionando como un asistente especializado disponible en todo momento.

Este proyecto combina una interfaz web sencilla y accesible con un modelo de lenguaje que procesa preguntas en tiempo real. Huddle AI está pensado para facilitar el acceso a datos relevantes de la NFL y mejorar la experiencia de los usuarios interesados en el deporte, ya sea desde una computadora o un dispositivo móvil.
